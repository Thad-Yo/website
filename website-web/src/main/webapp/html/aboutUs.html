集团简介
联信集团是目前国内网点最多的提供全面信用管理服务的集团公司
公司全称为青岛联信商务咨询有限公司，成立于1997年，注册资本为1500万人民币。于2004年开始操作信用卡逾期催收业务，现拥有14年的行业经验，其服务涉及金融和商业风险管理的各个环节，包括信用卡和消费信贷缴费提醒在内的金融外包服务，以及企业资信调查、投资调研、应收账款管理和呆坏账的催收与买卖业务。通过专业化的服务，我们帮助客户降低风险，改善现金流，实现长期稳定盈利。我司于2005年获得SGS颁发的ISO9001质量认证，成为中国大陆同行业中第一个获得此认证的公司。于2015年获得SGS颁发的ISO/IEC27001信息安全认证，以期用更加完善的体系为客户提供最优质的服务。
公司具有完善的流程、管理制度及严格的保密机制，确保银行等金融机构资料不被非法外泄。目前客户包括商业银行、国有银行、大型互联网金融机构以及部分小贷公司。
公司总部设在山东省青岛市，总部办公面积6000多平。除青岛总部外，联信集团已在全国另设42家分公司，全国范围内涵盖近百家网点，公司总办公面积31799多平，为全国范围内银行及其他金融机构的合作伙伴。现有员工近3477名，在职人员约70%为全日制本科学历，具备较强的谈判能力、解决问题的能力和良好的内外沟通能力，熟悉信贷业务、征信业务及相关法律知识，在职人员保持稳定。国范围内银行及其他金融机构的合作伙伴。
