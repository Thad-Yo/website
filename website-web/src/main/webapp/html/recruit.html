商务司机
7K-12K
温州市

职位要求

职责描述：

1、能够参与制订项目目标成本；

2、参与设计方案、施工方案优化，参与材料选型优化；

3、对项目动态成本及时反馈；

4、配合采购部进行工程及材料招标，完成工程及材料合同结算。

任职要求：

1、本科以上学历，工程管理/造价类相关专业，年龄30岁以下；

2、3年以上土建/安装造价、预决算工作经验，熟悉房地产成本管理体系和流程；

3、诚实正直，具有良好的沟通能力和协调能力；

4、持相关职业证书者优先。
